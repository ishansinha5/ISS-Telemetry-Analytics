# Sandbox: Conceptual Testing & Drafting

This directory serves as the development sandbox for the project. Before any logic is committed to the production `/scripts` pipelines, it is drafted, tested, and verified here in isolated Jupyter Notebook cells. 

This ensures that the core math—such as the Pearson correlations for orbital physics or the longitudinal time-shift heuristics for astrodynamics—is accurate before being wrapped into reusable, automated Python functions. 

*(Note: The notebooks in this folder are considered scratchpads and may contain rough, unoptimized code).*
